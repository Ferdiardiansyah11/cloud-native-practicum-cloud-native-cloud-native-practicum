### Latihan 3 — Tantangan: Bandingkan Routing

_Perbandingan antara React biasa dan Next.js terletak pada tingkat kemudahan dan kelengkapan fitur. React biasa memberi fleksibilitas tinggi, tetapi membutuhkan banyak konfigurasi manual seperti routing, setup build, serta integrasi backend yang terpisah. Sebaliknya, Next.js sudah menyediakan struktur yang lebih rapi dengan routing otomatis berbasis folder, konfigurasi minimal, serta dukungan backend melalui API routes dalam satu proyek. Akibatnya, proses pengembangan dengan Next.js cenderung lebih cepat dan sederhana dibandingkan React biasa yang lebih kompleks di tahap awal setup._
